Entity Framework Core model visualization

Get Started

No preinstall, zero config

Visualize Model layer of your application as a ER-like diagram without any 3-rd party tools. View the diagram in any browser, no plug-ins needed.

Better than the docs

Use database diagram instead of writing and constantly updating your documentation on the Model layer in your app. The diagram is perfect for both analysis and new features design. New developers in your project will love it, and your team meetings will be much more productive!

Collaborate

Use import/export to save your diagram as a JSON file and share it with your team.

Print

Print the diagram if you prefer working with paper. Will be useful on meetings.
